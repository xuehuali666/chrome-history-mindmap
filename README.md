# 浏览历史思维导图

Chrome 浏览器扩展，以思维导图形式展示浏览历史记录和自定义标签收藏。

## 功能

### 浏览历史
- 按主域名自动归类，时间从新到旧排序
- 点击域名展开/折叠，点击具体页面在新标签页打开
- 支持时间范围筛选（今天/7天/30天/3个月）

### 我的标签
- 将当前浏览页面收藏到自定义标签下
- 可选择已有标签或新建标签
- 支持自定义显示名称（直接回车使用网页标题）

### 设置
- 用网站名称代替域名显示（从网页标题自动提取）
- 去重方式：不去重 / 按网址去重 / 按标题去重
- 右键编辑域名节点的显示名称，日后相同域名自动归类
